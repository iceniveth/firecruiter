# Firecruiter

## Installation

Create a firebase project from firebase console.

Create ```.firebaserc``` file with:
```json
{
  "projects": {
    "default": "project-name"
  }
}
```

Install parcel bundler and dependencies.

```bash
npm install -g parcel-bundler
npm install
```

## Run Application

```bash
npm run dev
```

## Build Application
```bash
npm run build
```
